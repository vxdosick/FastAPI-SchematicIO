API Documentation for
SchematicIO

Integrate our AI-powered knowledge graph generation into your applications. Use the endpoints below to process text and generate interactive graphs.

Swagger Docs

Powered by FastAPI • Supports JSON and HTML responses

API Endpoints

Use these endpoints to generate knowledge graphs from text.

POST /api/schematic_io

Generate a knowledge graph from input text and return an HTML block for embedding.

Request (Form Data)

text: string (required) - The input text to analyze.

Response

HTMLResponse - An HTML string containing the Cytoscape.js graph embed.

Data Schemas

The following schemas define the structure of graph elements used in responses.

Node

{
  "id": "string",
  "label": "string"
}

Edge

{
  "id": "string",
  "source": "string",
  "target": "string"
}

Elements (Collection)

{
  "nodes": [Node],
  "edges": [Edge]
}

SchematicResponse

{
  "elements": Elements
}

This is the internal response format from the AI service, transformed into Cytoscape.js compatible elements.

Usage Example

How to call the API using curl.

curl -X POST "http://your-ngrok-url/api/schematic_io" \
  -F "text=Your article text here"

The response will be an HTML string you can embed in your page.
