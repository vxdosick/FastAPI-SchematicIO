API Endpoints

Use these endpoints to generate knowledge graphs from text.

A POST request to this endpoint.

POST /api/schematic_io_api

Generate a knowledge graph from input text and return an HTML block for embedding.

Request (Form Data)

text: string (required) - The input text to analyze.

Response

HTMLResponse - An HTML string containing the Cytoscape.js graph embed.

Data Schemas

The following schemas define the structure of graph elements used in responses.

SchematicResponse

{
"elements": Elements
}

Elements (Collection)

{
"nodes": [Node],
"edges": [Edge]
}

Node

{
"id": "string",
"label": "string"
}

Edge

{
"id": "string",
"source": "string",
"target": "string"
}

This is the internal response format from the AI service, transformed into Cytoscape.js compatible elements.

Usage Example

How to call the API using curl.

The response will be an HTML string you can embed in your page. 

curl -X 'POST' \
  'https://valorie-nonpsychiatric-gwendolyn.ngrok-free.dev/api/schematic_io_api' \
  -H 'Content-Type: application/json' \
  -d '{"text": "Python is a programming language used for web development and AI."}'
