Swagger Docs

Powered by FastAPI • Supports JSON and HTML responses

API Endpoints

Use these endpoints to generate knowledge graphs from text.

1. Request to receive the HTML block containing the diagram directly.

POST /api/schematic_io_block

- generate a knowledge graph from input text and return an HTML block for embedding.

Request (Form Data)

text: string

- this is a required field; please enter the title of the article for which you wish to create a diagram.

Response

JSONResponse - An HTML string containing the Cytoscape.js graph embed.

2. A request to retrieve the elements needed to create a block directly within Cytoscape.js.

POST /api/schematic_io_elements

- generation of key elements for creating a diagram in Cytoscape.js.

Request (Form Data)

text: string

- this is a required field; please enter the title of the article for which you wish to create an elements of a diagram.

Response

JSONResponse - A JSON object containing all the elements.

Data Schemas

The following schema defines the complete structure of graph elements used in responses.

POST /api/schematic_io_block

{
  "html_block": "string"
}

POST /api/schematic_io_elements

{
  "elements": {
    "nodes": [
      {
        "id": "string",
        "label": "string"
      }
    ],
    "edges": [
      {
        "id": "string",
        "source": "string",
        "target": "string"
      }
    ]
  }
}

This is the internal response format from the AI service, transformed into Cytoscape.js compatible elements.

Usage Example

How to call the API using curl.

1. The response will be an HTML string you can embed in your page. 

curl -X 'POST' \
  'https://valorie-nonpsychiatric-gwendolyn.ngrok-free.dev/api/schematic_io_block' \
  -H 'Content-Type: application/json' \
  -d '{"text": "Python is a programming language used for web development and AI."}'

2. The response will be an JSON object you can use in Cytoscape.js. 

curl -X 'POST' \
  'https://valorie-nonpsychiatric-gwendolyn.ngrok-free.dev/api/schematic_io_elements' \
  -H 'Content-Type: application/json' \
  -d '{"text": "Python is a programming language used for web development and AI."}'
